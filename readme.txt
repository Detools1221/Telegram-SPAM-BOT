-----------------МИНИ-ГАЙД---------------------
Папка sessions - для однопотока ( СЮДА НЕ БОЛЕЕ 1Й СЕССИИ )
Папка sessionsMulti - для многопотока ( СЮДА СКОЛЬКО УГОДНО СЕЙССИЙ )
text.txt - сюда текст рассылка
chats.txt - ссылки на чаты для автовхода
group.txt - чат для инвайтинга

Конвертер - input_session - сессию для конверта в tdata. output_tdata - там будет новая tdata. input_tdata - tdata для конверта в session. в output_session будет новая сессия
Для парсинга пользователей из комментариев - введите ссылки на посты в linksforparsing.txt в столбик. НЕ ПАРСИТ С ЗАКРЫТЫХ КАНАЛОВ! 
Когда спарсили пользователей, они появятся в файле members.csv, из него же берутся люди для рассылки в лс, инвайтинга

Обязательно использовать прокси - так ваши аккаунты проживут дольше

settings.txt - файл с настройками, если меняете что-то в нем, то лучше перед этим сохранить копию.

-------------SYNTAX-----------------
Вы можете заменить стандартные символы для замены в файле syntax.txt, для этого удалите стандартные символы для замены и впишите свои, например a:α:А:Ẵ - a:X:А:Y - меняются символы, которые обозначены X Y!

-----------RANDOMIZER---------------
Рандомизация текста - замена одних символов, похожими, чтобы телеграм не думал, что вы отправляете одно и то же сообщение.

в многопотоке пока что нельзя изменить значение рандомизации, оно выбирается рандомно от 300 до 700

Работает только с рассылкой по чатам однопоток и многопоточной рассылкой по чатам.
Сообщение, которое вы отправляете - берется из text.txt, а на которое хотите заменить - путем копирования из канала, для этого -
вам нужно в group.txt вставить ссылку на канал, откуда будет взят текст, при этом канал должен быть открытый
